# Jokenpô 

## Descrição  
Jogo interativo de Jokenpô (Pedra, Papel e Tesoura) onde o usuário compete contra a máquina, testando sorte e estratégia. Desenvolvido para aprimorar habilidades em JavaScript, HTML e CSS, com foco em lógica de programação e manipulação de eventos no DOM.

## Tecnologias Utilizadas  

- HTML5 para estruturação da interface  

- CSS3 para estilização simples e responsiva  

- JavaScript para lógica do jogo e interatividade

## Funcionalidades  

- Interface simples, intuitiva e responsiva  

- Escolha entre Pedra, Papel ou Tesoura com clique do usuário  

- Computador faz escolhas aleatórias com lógica programada  

- Feedback instantâneo do resultado da rodada (vitória, derrota ou empate)  

- Contador de placar para acompanhar desempenho

## Como Jogar  

1. Escolha uma opção: Pedra, Papel ou Tesoura  

2. Veja a escolha do computador e o resultado da partida  

3. Repita para jogar quantas vezes quiser e testar sua sorte!
